# Skin Disease Classification using Transfer Learning

Deep learning model to classify 7 skin diseases using ResNet-50 on HAM10000 dataset.

## Results
- Accuracy: 89%
- F1 Score: 0.87
- Dataset: HAM10000 (10,015 images)

## How to Run
pip install -r requirements.txt
python train.py
python predict.py your_image.jpg

## Tech Stack
Python, PyTorch, ResNet-50, Gradio
